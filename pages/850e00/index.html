# 现在的Kubernetes

Kubernetes 成为了事实上的云原生操作系统，是容器编排领域的王者至尊。

毕竟，现代应用是什么？是微服务，是服务网格，这些统统要围绕着容器来开发、部署和运行，而使用容器就必然要用到容器编排技术，在现在只有唯一的选项，那就是Kubernetes。

Docker和Kubernetes用的都是Go。

WHY USE DOCKER ? 要在客户的环境里部署自研应用，但依赖库差异太大，很难搞定，但用容器会方便很多。
